# Markdown 

> Markdown 的目标是实现「易读易写」。  
>
> Markdown是一种轻量级标记语言，您可以使用它将格式化元素添加到明文文本文档中。Markdown于2004年由John Gruber创建，现在是全球最受欢迎的标记语言之一。
